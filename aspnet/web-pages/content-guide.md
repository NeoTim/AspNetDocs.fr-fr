# <a name="webmatrix-content-guide"></a>Guide de contenu de WebMatrix

> [!NOTE] 
> WebMatrix n’est plus recommandé comme environnement de développement intégré pour ASP.NET pages Web. Utilisez [Visual Studio](xref:web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) ou Visual Studio [Code](https://code.visualstudio.com/).

WebMatrix est un ensemble d’outils qui fournit un moyen facile de construire des sites Web pour Microsoft Windows. Il comprend IIS Express (un serveur Web de développement), ASP.NET (un cadre de programmation web) et SQL Server Compact (une base de données intégrée). Pour voir une visite rapide de WebMatrix, voir [WebMatrix Aperçu](https://www.microsoft.com/web/webmatrix/). Pour consulter les principales fonctionnalités webMatrix, voir [WebMatrix Features](https://www.microsoft.com/web/webmatrix/features/).
