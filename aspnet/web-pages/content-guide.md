# <a name="webmatrix-content-guide"></a>Guide de contenu de WebMatrix

> [!NOTE] 
> WebMatrix n’est plus recommandé en tant qu’environnement de développement intégré pour pages Web ASP.NET. Utilisez [Visual Studio](xref:aspnet/web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) ou [Visual Studio code](https://code.visualstudio.com/).

WebMatrix est un ensemble d’outils qui offre un moyen simple de créer des sites Web pour Microsoft Windows. Il comprend IIS Express (un serveur Web de développement), ASP.NET (une infrastructure de programmation Web) et SQL Server Compact (une base de données incorporée). Pour obtenir une présentation rapide de WebMatrix, consultez [vue d’ensemble de WebMatrix](https://www.microsoft.com/web/webmatrix/). Pour passer en revue les principales fonctionnalités WebMatrix, consultez [fonctionnalités WebMatrix](https://www.microsoft.com/web/webmatrix/features/).
