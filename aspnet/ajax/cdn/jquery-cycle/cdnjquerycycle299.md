# <a name="jquery-cycle-299"></a>jQuery Cycle 2.99

Les fichiers de cycle jQuery suivants sont hébergés sur ce CDN :

- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.js) &mdash; la version complète du plug-in de cycle jQuery. Comprend plus de deux douzaines d’effets spéciaux et toutes les options.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.all.min.js) &mdash; la version minimisés du fichier ci-dessus. Utilisez cette version pour les applications de production afin d’améliorer les performances de votre site Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.js)&mdash; cette version du plug-in de cycle jQuery est plus petite et contient uniquement l’effet de transition « fondu ».
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.min.js) &mdash; la version minimisés du fichier ci-dessus. Utilisez cette version pour les applications de production afin d’améliorer les performances de votre site Web.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.js) &mdash; une version encore plus petite du plug-in de cycle jQuery qui contient uniquement la transition « fondu » sans toutes les options.
- [https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js](https://ajax.aspnetcdn.com/ajax/jquery.cycle/2.99/jquery.cycle.lite.min.js) &mdash; la version minimisés du fichier ci-dessus. Utilisez cette version pour les applications de production afin d’améliorer les performances de votre site Web.
