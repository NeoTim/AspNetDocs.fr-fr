<span data-ttu-id="60b87-153">Chaque fois que vous acceptez l’entrée d’un utilisateur et réaffichez cette entrée sur votre site Web, vous ouvrez potentiellement votre site Web à des attaques de Cross-Site Scripting (XSS).</span><span class="sxs-lookup"><span data-stu-id="60b87-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="60b87-154">En théorie, un pirate pourrait envoyer le code JavaScript qui est exécuté quand l’entrée est réaffichée.</span><span class="sxs-lookup"><span data-stu-id="60b87-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="60b87-155">Le code JavaScript peut être utilisé pour voler les mots de passe utilisateur ou d’autres informations sensibles.</span><span class="sxs-lookup"><span data-stu-id="60b87-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>
