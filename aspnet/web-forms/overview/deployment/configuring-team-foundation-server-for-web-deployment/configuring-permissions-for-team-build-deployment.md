## <a name="configuring-web-server-permissions"></a><span data-ttu-id="937f0-119">Configuration des autorisations de serveur Web</span><span class="sxs-lookup"><span data-stu-id="937f0-119">Configuring Web Server Permissions</span></span>

<span data-ttu-id="937f0-120">Comme décrit dans [la rubrique choix de l’approche appropriée pour le déploiement Web](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), il existe deux approches principales que vous pouvez utiliser si vous souhaitez déployer des packages Web sur un serveur Web distant :</span><span class="sxs-lookup"><span data-stu-id="937f0-120">As described in [Choosing the Right Approach to Web Deployment](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), there are two main approaches you can use if you want to deploy web packages to a remote web server:</span></span>

<span data-ttu-id="937f0-129">En revanche, l’approche du gestionnaire de Web Deploy offre différents avantages :</span><span class="sxs-lookup"><span data-stu-id="937f0-129">In contrast, the Web Deploy Handler approach offers various advantages:</span></span>
