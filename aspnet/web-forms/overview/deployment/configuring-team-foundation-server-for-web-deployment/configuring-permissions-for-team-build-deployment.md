## <a name="configuring-web-server-permissions"></a><span data-ttu-id="369c2-119">Configuration des autorisations de serveur Web</span><span class="sxs-lookup"><span data-stu-id="369c2-119">Configuring Web Server Permissions</span></span>

- <span data-ttu-id="369c2-121">Déployer l’application à partir d’un emplacement distant en ciblant le *Service de l’Agent de déploiement Web* (également appelé l’agent distant) sur le serveur de destination.</span><span class="sxs-lookup"><span data-stu-id="369c2-121">Deploy the application from a remote location by targeting the *Web Deployment Agent Service* (also known as the remote agent) on the destination server.</span></span>
