# <a name="how-do-i-use-the-aspnet-ajax-tabs-control"></a>[Comment faire :] Utiliser le contrôle onglets AJAX ASP.NET ?

par [Joe Stagner](https://github.com/JoeStagner)

Cette vidéo montre les principes de base du contrôle onglets de la boîte à outils de contrôle ASP.NET AJAX, avec lequel vous pouvez créer un ensemble d’onglets pour organiser le contenu sur une page Web. L’onglet sélectionné est conservé dans les publications, ce qui rend le contrôle des onglets idéal pour les pages d’administration basées sur des formulaires.
