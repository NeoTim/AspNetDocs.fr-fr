# <a name="how-do-i-use-a-hidden-field-to-store-and-manipulate-client-side-information"></a>[Comment faire :] Utiliser un champ masqué pour stocker et manipuler des informations côté client

par [Chris pixels](https://twitter.com/chrispels)

Dans cette vidéo, Chris pixels montre comment utiliser un champ masqué comme une approche de la gestion d’état simple sur une page ASP.NET et pour communiquer les informations de traitement côté client. Tout d’abord, consultez Comment ajouter un champ masqué à une page à l’aide d’un contrôle serveur ASP.NET HiddenField et récupérer la valeur du champ masqué dans le code côté serveur après une publication. Ensuite, Découvrez comment manipuler la valeur du champ masqué dans le script côté client, puis récupérer la valeur modifiée dans le code côté serveur. Enfin, consultez un exemple d’utilisation de plusieurs champs masqués pour stocker des informations provenant du traitement côté client qui peuvent être publiées pour le traitement du serveur.
