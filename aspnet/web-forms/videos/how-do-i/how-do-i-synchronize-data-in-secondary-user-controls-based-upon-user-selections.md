# <a name="how-do-i-synchronize-data-in-secondary-user-controls-based-upon-user-selections"></a>[Comment faire] : synchroniser les données dans le ou les contrôles utilisateur secondaires en fonction des sélections de l’utilisateur

par [Chris pixels](https://twitter.com/chrispels)

Dans cette vidéo, Chris pixels montre comment créer deux contrôles DropDownList où la sélection d’un élément dans la première liste remplit les éléments de la deuxième liste. Tout d’abord, une page ASP.NET est créée avec deux contrôles DropDownList, l’un pour le client et l’autre pour les États dans lesquels le client fait des affaires. La liste des clients est ensuite renseignée dans le code-behind. Ensuite, dans l’événement SelectedIndexChanged de la liste Customer, les éléments de la liste State sont renseignés en fonction de la sélection. Enfin, il existe une discussion sur la façon dont cette approche peut être appliquée à d’autres types de contrôles et de données.
