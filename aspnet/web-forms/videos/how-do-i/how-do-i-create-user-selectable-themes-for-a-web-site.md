# <a name="how-do-i-create-user-selectable-themes-for-a-web-site"></a>[Comment faire :] Créer des thèmes sélectionnables par l’utilisateur pour un site Web

par [Chris pixels](https://twitter.com/chrispels)

Dans cette vidéo, Chris pixels montre comment créer plusieurs thèmes pour un site Web, puis autoriser l’utilisateur à sélectionner le thème qui est appliqué au site. Découvrez comment créer une interface utilisateur simple qui permet à l’utilisateur de sélectionner chacun des thèmes avec un contrôle utilisateur. En outre, Découvrez comment définir un thème pour une page de contenu à partir d’un contrôle utilisateur sur une page maître.
