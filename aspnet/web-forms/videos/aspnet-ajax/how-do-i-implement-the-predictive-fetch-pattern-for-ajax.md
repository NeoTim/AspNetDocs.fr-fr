# <a name="how-do-i-implement-the-predictive-fetch-pattern-for-ajax"></a>[Comment faire :] Implémenter le modèle d’extraction prédictive pour AJAX ?

par [Joe Stagner](https://github.com/JoeStagner)

Illustre une implémentation du modèle d’extraction prédictif pour AJAX, où le moteur AJAX utilise l’état actuel de l’interaction de l’utilisateur pour prédire quel jeu de contenu sera requis ensuite, puis récupère ces données à l’avance à partir du serveur. Vous bénéficiez ainsi d’une expérience utilisateur plus réactive.
