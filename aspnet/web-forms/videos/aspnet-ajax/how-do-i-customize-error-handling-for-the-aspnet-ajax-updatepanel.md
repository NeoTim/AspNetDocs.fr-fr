# <a name="how-do-i-customize-error-handling-for-the-aspnet-ajax-updatepanel"></a>Comment faire : personnaliser la gestion des erreurs pour ASP.NET AJAX UpdatePanel

par [Rob Windsor](https://twitter.com/robwindsor)

Lorsqu’une erreur se produit pendant le traitement d’une publication (postback) déclenchée par un UpdatePanel ASP.NET AJAX, le PageRequestManager côté client affiche une alerte (.NET 2,0 ou 3,0) ou lève de nouveau l’exception (.NET 3,5 ou version ultérieure). Dans les deux cas, l’expérience de l’utilisateur final est inférieure à ce qui est optimal. Dans cette vidéo, Rob Windsor va vous montrer comment résoudre ce problème en personnalisant la gestion des erreurs associées à UpdatePanel.
