# <a name="how-do-i-use-the-aspnet-ajax-timer-control"></a>[Comment faire :] Utiliser le contrôle Timer ASP.NET AJAX ?

par [Joe Stagner](https://github.com/JoeStagner)

Le contrôle Timer AJAX permet à une partie d’une page Web ASP.NET d’être mise à jour de manière dynamique à intervalles réguliers, plutôt que d’avoir besoin de l’utilisateur pour effectuer une action telle qu’un clic sur un bouton.
