# <a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[Comment faire :] Utiliser le contrôle UpdateProgress ASP.NET AJAX ?

par [Joe Stagner](https://github.com/JoeStagner)

Dans une application AJAX, il y a souvent une pause perceptible entre le moment où le navigateur effectue une requête réseau et le moment où le navigateur reçoit une réponse. Pour informer l’utilisateur que la conversation réseau est en cours, le contrôle ASP.NET AJAX UpdateProgress fournit des commentaires immédiats en affichant une image animée, en « chargeant » du texte ou tout autre code HTML qui disparaît une fois la réponse réseau reçue.
