# <a name="how-do-i-use-generic-tests"></a>[Comment faire :] Utiliser des tests génériques ?

par [Chris Menegay](https://twitter.com/CMenegay)

Un test générique est un programme existant encapsulé et mis à disposition en tant que test dans Visual Studio Team System. Dans cette vidéo, nous voyons une démonstration de la façon dont un programme de Shell encapsulé accepte une valeur d’entrée de notre environnement de test et comment la valeur de sortie du programme est utilisée pour déterminer si le test a réussi ou échoué.
