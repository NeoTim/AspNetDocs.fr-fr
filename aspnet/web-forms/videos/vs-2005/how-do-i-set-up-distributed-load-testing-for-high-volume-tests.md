# <a name="how-do-i-set-up-distributed-load-testing-for-high-volume-tests"></a>[Comment faire :] Configurer des tests de charge distribués pour des tests de volume élevés ?

par [Chris Menegay](https://twitter.com/CMenegay)

Visual Studio Team System vous permet de configurer des tests de charge distribuée en créant un test « plate-forme » qui comprend un contrôleur sur un ordinateur distant et plusieurs agents sur des ordinateurs distants identiques ou supplémentaires. Cette vidéo vous guide tout au long de l’installation du contrôleur et des agents, puis configuration les cas de test à exécuter à l’aide du contrôleur distant. Comme vous le verrez, Visual Studio Team System simplifie la configuration des tests de charge distribués pour les tests automatisés à volume élevé.
