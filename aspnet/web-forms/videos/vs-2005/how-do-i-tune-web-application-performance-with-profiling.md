# <a name="how-do-i-tune-web-application-performance-with-profiling"></a>[Comment faire :] Paramétrer les performances des applications Web avec le profilage ?

par [Chris Menegay](https://twitter.com/CMenegay)

Dans cette vidéo, nous apprenons à Visual Studio Team System qui propose deux méthodes de profilage d’une application Web : l’échantillonnage et l’instrumentation. L’instrumentation injecte des marqueurs dans le code sous-jacent afin que le développeur puisse afficher les détails complets de l’exécution de l’application, y compris les arborescences des appels et l’allocation des objets, afin d’optimiser les performances de l’application.
