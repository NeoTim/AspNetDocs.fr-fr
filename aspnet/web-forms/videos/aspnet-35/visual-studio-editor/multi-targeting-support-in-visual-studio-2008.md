# <a name="multi-targeting-support-in-visual-studio-2008"></a>Prise en charge du multiciblage dans Visual Studio 2008

par [Joe Stagner](https://github.com/JoeStagner)

Je vais illustrer la fonctionnalité multicible de Visual Studio 2008. Dans le passé, vous deviez choisir la version de Visual Studio pour qu’elle corresponde à la version .NET ciblée. Par exemple, Visual Studio 2005 fonctionnait avec .NET 2,5. Désormais, avec Visual Studio 2008, vous pouvez choisir la version de .NET adaptée à votre application.
