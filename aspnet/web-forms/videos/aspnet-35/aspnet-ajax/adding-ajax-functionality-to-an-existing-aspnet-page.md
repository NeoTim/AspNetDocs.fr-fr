# <a name="adding-ajax-functionality-to-an-existing-aspnet-page"></a>Ajout de la fonctionnalité AJAX à une page ASP.NET existante

par [Todd Miranda](https://github.com/tmiranda)

L’utilisation d’AJAX dans les pages ASP.NET peut augmenter les performances et améliorer l’expérience utilisateur globale d’une application. Il n’est pas nécessaire de réécrire les pages existantes pour ajouter des fonctionnalités AJAX. Avec quelques modifications seulement, une page ASP.NET existante peut être créée pour utiliser AJAX. Cette vidéo montre comment ajouter des fonctionnalités AJAX à une page ASP.NET existante avec un minimum d’effort.
