# <a name="how-do-i-introducing-the-new-chart-control-in-visual-studio-2010"></a>Comment faire : présentation du nouveau contrôle Chart dans Visual Studio 2010

par Adams max.

Il y a beaucoup d’enthousiasme pour la version imminente de Microsoft Visual Studio 2010 et la .NET Framework 4,0. Parmi les nombreuses nouvelles fonctionnalités, on trouve le contrôle de graphique Web ASP.NET. Dans cette vidéo, Max Adams présente le contrôle et démontre combien il est facile de créer des outils d’analyse financière ou financière visuellement attrayants dans un navigateur à l’aide du contrôle Chart.
