# <a name="how-do-i-use-routing-with-aspnet-web-forms"></a>Comment faire : utiliser le routage avec ASP.NET Web Forms ?

par [Chris pixels](https://twitter.com/chrispels)

Dans cette vidéo, Chris pixels montre comment implémenter le routage pour Web Forms dans ASP.NET 4. Tout d’abord, le concept de routage d’une URL est comparé au mappage de l’URL à un fichier physique dans le site. Ensuite, un exemple d’itinéraire pour une URL est défini dans le fichier global. asax application\_gestionnaire d’événements de démarrage. L’itinéraire contient une valeur paramétrable que l’utilisateur peut entrer dans l’URL. Une page d’exemple est ensuite créée et la valeur du paramètre d’itinéraire est extraite dans la page\_gestionnaire d’événements de chargement. Ensuite, un deuxième itinéraire avec plusieurs paramètres et itinéraires vers la même page que l’itinéraire initial est défini. Le gestionnaire d’événements de chargement de page\_est développé pour extraire la valeur de paramètre d’itinéraire supplémentaire et afficher des informations différentes selon les valeurs qui ont été passées à la page.
