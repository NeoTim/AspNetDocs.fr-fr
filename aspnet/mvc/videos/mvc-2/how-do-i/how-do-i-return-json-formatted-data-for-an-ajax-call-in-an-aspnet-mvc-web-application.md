# <a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>Comment faire : retourner des données au format JSON pour un appel AJAX dans une application Web ASP.NET MVC ?

par [Chris pixels](https://twitter.com/chrispels)

Dans cette vidéo, Chris pixels montre comment utiliser les classes JsonResult et JSON pour retourner des instances de classes sous forme de données au format JSON. Tout d’abord, un exemple d’application Web MVC est créé et un contrôleur et une vue associée sont ajoutés pour les informations client. Une classe Customer simple est ensuite créée dans le cadre du modèle de données. Une fonction GetCustomer () est ensuite ajoutée au contrôleur du client pour récupérer les données d’un seul client au format JSON. Enfin, un script est ajouté à la vue client qui utilise jQuery pour effectuer un appel AJAX à GetCustomer () afin de récupérer les données client au format JSON
